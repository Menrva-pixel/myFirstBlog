Introduction

Website ini dibuat untuk memenuhi tugas / assigment kelas belajar dasar web.

Perkenalan

ini adalah website blog pribadi yang dibuat khusus untuk mencatat kegiatan selama belajar di platform dicoding academy.

Website ini dibuat untuk memenuhi tugas praktek pertama di platform belajar Dicoding.org dan website ini akan terisi dengan catatan kegiatan selama saya belajar di platform dicoding academy.

Mengapa Dicoding?

Menurut saya Pribadi dicoding.org adalah plaform belajar terbaik khususnya di bidang IT, saya pribadi sudah sangat ingin bergabung di platform belajar dicoding semenjak 2020, namun baru kesampaian di tahun 2023 ini, karna ajakan dan beberapa refrensi dari teman kuliah, saya jadi makin penasaran.
